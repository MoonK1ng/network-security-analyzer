# Network Security Analyzer

Aplikacja do analizy bezpieczeństwa protokołów sieciowych (HTTP, HTTPS, DNS, FTP, SMTP).

## Instalacja

```bash
pip install -r requirements.txt
```

## Uruchomienie

```bash
python main.py
```

## Funkcje

- Przechwytywanie pakietów sieciowych
- Analiza protokołów: HTTP, HTTPS, DNS, FTP, SMTP
- Wykrywanie niezaszyfrowanych danych
- Identyfikacja słabych protokołów
- Wykrywanie podejrzanej aktywności
- Generowanie raportów bezpieczeństwa
- GUI interfejs

## Wymagania

- Python 3.8+
- Uprawnienia administratora (do przechwytywania pakietów)
- Windows: Npcap lub WinPcap
